## Quick Start

### clone github repository
```bash
git clone https://github.com/Team-SMDRS/MicroBankingSystem-DB-Frontend
```

### install dependencies
```bash
cd MicroBankingSystem-DB-Frontend
npm install
```

### run the app
```bash
npm run dev
```
